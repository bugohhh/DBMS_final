Demo 流程

  1. 輸入 machine_id，例如 1。
  2. 按「刷新庫存」讀取資料庫庫存。
  3. 按商品的「購買」按鈕。
  4. 後端新增 SalesRecord，並扣除 Inventory.quantity。
  5. 畫面立即更新，不需手動刷新。
  6. 若後台補貨完成，按「刷新庫存」重新同步。
